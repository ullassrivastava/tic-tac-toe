# Tic-Tac-Toe
## Introduction
Simple game of tic tac toe with an AI

## Technologies Used:
Front End :- HTML5, CSS, JavaScript<br/>

## Instructions
Click on any boxes to put your 'X' mark. Opponent will put his 'O' mark. The first one to make the line of 3 wins.
